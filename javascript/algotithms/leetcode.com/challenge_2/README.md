# Two Sum

**Link: https://leetcode.com/problems/palindrome-number/
Difficulty:** Easy

## Hint

Given an integer `x`, return `true` if _`x`_ is a \***\*palindrome\*\***, and _`false` otherwise_ .

## Constraints:

- `-2^31 <= x <= 2^31` ` - 1`

## Follow-up:

**Follow up:** Could you solve it without converting the integer to a string?
